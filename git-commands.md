# Git Initialization & First Push Guide

Follow these steps to initialize Git and push the boilerplate to your GitHub repository:

```bash
# 1. Initialize local repository
git init -b main

# 2. Add all files
git add .

# 3. Create initial commit
git commit -m "chore: initial project setup for HanGan Monorepo"

# 4. Add your GitHub remote (replace with your actual repository URL)
git remote add origin https://github.com/Vongpasith/HanGan.git

# 5. Push to the main branch
git push -u origin main
```
